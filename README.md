### 🛠️ Projects

> Repositories are private — live demos available on request.

#### 💰 Lutu'uria (Lutu) — Financial Management PWA
`Next.js` `React` `TypeScript` `Tailwind CSS` `Prisma` `PostgreSQL` `NextAuth` `IndexedDB` `Vercel`

Adaptive PWA (Desktop/Mobile) for managing loan portfolios at **Sonora Forge**.
- **Ledger system** with precise separation of capital, interest, late fees and reinvestment
- **Client file engine** that tracks payment history, incidents and overdue installments with WhatsApp follow-up integration
- **Multi-auth flows**: Google OAuth + OTP via email (Resend) + 6-digit PIN with bcrypt
- **Multi-format exports**: PDF, Excel, CSV and JSON account statements
- **Partial offline support** via IndexedDB + Service Worker + Web App Manifest

---

#### 📋 Gestión FMS — Internal ERP/CRM
`Next.js 16` `React 19` `TypeScript` `Tailwind CSS` `shadcn/ui` `Prisma 6` `PostgreSQL` `NextAuth v5`

Operational ERP/CRM for **Felix MultiServicios** technical services.
- **Full service workflow**: diagnostics, spare parts management, field technician assignment and debt tracking
- **Role-based interfaces** for Administrator and Technician profiles with strict access control
- **Transactional data model**: cash register cuts, inventory reservations and audit trail with Prisma 6 + Neon
- **2FA authentication** with NextAuth v5 beta, JWT, bcrypt and protected Server Actions
